# Czech Academic Corpus / Český akademický korpus

This is the working repository where annotation errors can be fixed.

Local (ÚFAL): /net/projects/cac holds the official releases 1.0 and 2.0.

CAC is converted to [Universal Dependencies](https://github.com/UniversalDependencies/UD_Czech-CAC)
and regularly released in UD. Any errors found in the UD version must be fixed either upstream
(here) or in the [conversion code](https://github.com/ufal/hamledt/tree/master/normalize/cs-cac).
